## 🚀 How to Deploy (GitHub Pages — Free Hosting)

### Step 1 — Create a GitHub Account
If you don't already have one, sign up at [github.com](https://github.com).

### Step 2 — Create a New Repository
1. Click the **+** icon in the top right → **New repository**
2. Name it something like `locdbyroe-website`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload Your Files
1. Click **Add file** → **Upload files**
2. Upload `locdbyroe.html` — then **rename it to `index.html`** before uploading, or rename it after
3. Upload this `README.md` file
4. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo → **Settings**
2. Scroll down to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch → **/ (root)** folder
5. Click **Save**

### Step 5 — Your Site is Live! 🎉
Within 1–2 minutes your site will be live. GitHub will show you the URL at the top of the Pages settings page.

## ✏️ How to Make Changes

Since this is a single HTML file, all edits happen inside `index.html`.

### To update text or contact info:
1. Open `index.html` in any text editor (Notepad, VS Code, etc.)
2. Use **Find** (`Ctrl+F` / `Cmd+F`) to search for the text you want to change
3. Edit it, save the file
4. Re-upload to GitHub (it will replace the old file)
